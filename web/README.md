## Rowan Carter · Frontend Engineer

A single-page portfolio highlighting a three-year React and Angular engineer. The site blends React-driven interactivity with an embedded Angular runtime widget to demonstrate cross-framework fluency.

### Tech Stack

- **Next.js 14** with the App Router
- **React 18** client and server components
- **Angular 17 (JIT)** bootstrapped at runtime via CDN
- **Tailwind CSS v4** with radial-gradient theming

### Available Scripts

- `npm run dev` – start the local development server
- `npm run build` – create a production build
- `npm start` – serve the production build
- `npm run lint` – lint the project via ESLint

### Deployment

Deploy directly to Vercel:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-8c43cf6f
```
